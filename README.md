# Sushida Typing Bot (寿司打自動化ボット)

「寿司打 (https://sushida.net/)」を自動でプレイするPythonボットです。
画面をキャプチャして文字を認識し、自動でタイピングを行います。

## 機能

- **Selenium**: ブラウザを自動で開き、ゲームページへアクセスします。
- **MSS**: 高速なスクリーンキャプチャを行います。
- **EasyOCR**: 画面上のローマ字を認識します。
- **PyAutoGUI**: 認識した文字をキーボード入力として送信します。

## 必要条件

- Windows OS
- Python 3.8 以上
- Google Chrome ブラウザ

## インストール方法

1. リポジトリをダウンロード（またはクローン）します。
2. フォルダ内でコマンドプロンプトを開き、必要なライブラリをインストールします。

```bash
pip install -r requirements.txt
```

※ 初回実行時にOCRモデルのダウンロードが行われるため、時間がかかる場合があります。

## 使い方

簡単起動用のバッチファイルを用意しています。

1. **`run_bot.bat`** をダブルクリックして実行してください。
2. ブラウザが立ち上がり、寿司打のページが開きます。
3. 以下の手順でゲームを開始してください：
   - ゲームモード（3000円コース・10000円コース等）を選択
   - カウントダウンが終わり、ゲーム画面が表示されるのを待つ
   - コンソール（黒い画面）で **Enterキー** を押すと、自動入力が開始されます。

停止する場合は、コンソール画面で `Ctrl+C` を押すか、ウィンドウを閉じてください。

## トラブルシューティング

- **文字認識がうまくいかない場合**: ブラウザのウィンドウがメインモニターの中央にあることを確認してください。
- **動作が重い場合**: GPUを使用しない設定になっています（安定性のため）。PCスペックによっては遅延が発生する場合があります。

## ディレクトリ構成

- `logs/`: 実行ログが保存されます。
- `debug/`: デバッグ用のスクリーンショットが保存されます（`.gitignore`によりGitHubにはアップロードされません）。
